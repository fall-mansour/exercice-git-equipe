# Bilan Git - Le Pilote 

Voici les trois compétences clés que j'ai acquises durant les Labs Git 1, 2 et 3 :

1.**Gestion des conflits :** J'ai appris qu'un conflit n'est pas une erreur, mais un événement normal qui survient lorsque deux membres modifient la même ligne d'un fichier. Je sais désormais lire les marqueurs `<<<<<<< HEAD`, `=======` et `>>>>>>>` pour choisir la version finale.

2. **Maîtrise du Workflow Collaboratif :** Je maîtrise le cycle complet de développement : créer une branche `feature`, coder, effectuer un `push`, puis ouvrir une Pull Request (PR) pour que l'Architecte puisse valider et merger mon code dans `main`.

3. **Bonnes pratiques et Sécurité :** Je sais utiliser le fichier `.gitignore` pour empêcher Git de suivre des fichiers sensibles comme le `.env` (clés API) ou les dossiers de dépendances.
J'ai aussi appris à poser des tags de version (ex: `v0-git-termine`) pour marquer les jalons importants du projet.
