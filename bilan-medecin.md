# Bilan Labs Git — Le Médecin

## 3 choses apprises dans les Labs Git

1. Les branches permettent de travailler en parallèle sans toucher au travail des autres membres.
2. Un conflit Git survient quand deux personnes modifient la même ligne du même fichier — ce n'est pas une erreur, on le résout calmement en lisant les marqueurs.
3. Le workflow complet (branche → commit → push → Pull Request → merge → tag) est la base de toute collaboration Git professionnelle.
